# ⚙️ Auto Dashboard

Painel web para monitoramento de sistema e execução de automações.

## 🚀 Funcionalidades

* Monitoramento de CPU, memória e uptime
* Execução de automações simuladas
* Interface simples e moderna

## 🛠️ Tecnologias

* Node.js (Express)
* HTML, CSS, JavaScript

## 💡 Objetivo

Simular um ambiente real de controle de automações e infraestrutura.

## 📌 Futuras melhorias

* Integração com APIs reais (n8n, bots)
* Autenticação de usuários
* Logs em tempo real
* Deploy com Cloudflare Tunnel
